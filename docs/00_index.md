## 最重要目的

- このプロジェクトの目的は、LLMを使用して **人間の人格を再現すること** である
- 感覚器の数や処理性能は人間と同一でなくてよく、重要なのは「それらをどう人格として統合し、どう振る舞うか」である
- 実装や設計を読むときは、「その変更が人間らしい人格・記憶・感情・振る舞いの再現にどう効くか」を最優先で判断する
