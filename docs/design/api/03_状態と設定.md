# 状態と設定

## 状態面

### `GET /api/status`

- 認証: 必要
- 役割: 運用に必要な設定スナップショットとランタイム要約を返す

response:

```json
{
  "ok": true,
  "data": {
    "settings_snapshot": {
      "selected_persona_id": "persona:default",
      "selected_memory_set_id": "memory_set:default",
      "desktop_watch": {
        "enabled": false,
        "interval_seconds": 300
      },
      "wake_policy": {
        "mode": "disabled"
      },
      "selected_model_preset_id": "model_preset:default"
    },
    "runtime_summary": {
      "connection_state": "ready",
      "wake_scheduler_active": false,
      "ongoing_action_exists": false,
      "memory_job_worker_active": true,
      "pending_memory_job_count": 0,
      "memory_job_in_progress": false
    }
  }
}
```

- `wake_scheduler_active` は、server 内の background 起床スケジューラが稼働中で、かつ `wake_policy.mode=interval` のとき `true` を返す
- `memory_job_worker_active` は、embedding 更新と `reflective consolidation` を処理する background worker が稼働中のとき `true` を返す
- `pending_memory_job_count` は、未完了の記憶後段 job 数を返す
- `memory_job_in_progress` は、worker が現在 1 件の job を処理中のとき `true` を返す

## 設定面

### `GET /api/config`

- 認証: 必要
- 役割: 現在設定と現在選択中の設定定義を返す

response:

```json
{
  "ok": true,
  "data": {
    "settings_snapshot": {
      "selected_persona_id": "persona:default",
      "selected_memory_set_id": "memory_set:default",
      "desktop_watch": {
        "enabled": false,
        "interval_seconds": 300
      },
      "wake_policy": {
        "mode": "disabled"
      },
      "selected_model_preset_id": "model_preset:default"
    },
    "selected_persona": {},
    "selected_memory_set": {},
    "selected_model_preset": {}
  }
}
```

- `selected_model_preset.roles.*.api_key` と `selected_memory_set.embedding.api_key` の実値は返さない
- 通常の読み取り系では、秘密値の有無を `api_key_present` で返す
- ただし `editor-state` は編集 bundle なので、`model_presets.roles.*.api_key` と `memory_sets.embedding.api_key` を含む

### `PATCH /api/config/current`

- 認証: 必要
- 役割: 現在設定のうち変更したい項目だけを更新する
- 対象は `selected_persona_id`、`selected_memory_set_id`、`selected_model_preset_id`、`desktop_watch`、`wake_policy` に限る
- 設定定義そのものはこの endpoint では変えない
- `selected_memory_set_id` は人格設定と独立に変更する

request body の最小 shape:

```json
{
  "selected_persona_id": "persona:default",
  "selected_memory_set_id": "memory_set:default",
  "selected_model_preset_id": "model_preset:default",
  "desktop_watch": {
    "enabled": false,
    "interval_seconds": 300
  },
  "wake_policy": {
    "mode": "disabled"
  }
}
```

request body には変更する項目だけを含める。response は `GET /api/config` と同じ shape を返す。
この endpoint は current 設定スナップショットだけを対象にした補助 patch 面であり、任意の複数資源をまたぐ汎用 transaction 面にはしない。

### `PUT /api/config/editor-state`

- 認証: 必要
- 役割: 現在設定と設定定義群を bundle 単位で全体置換する
- `current`、`personas`、`memory_sets`、`model_presets` を 1 つの editor bundle として検証し、部分更新しない
- 通常の意味単位操作とは別に、`CocoroConsole` の編集 UI 用の明示的な bundle 操作として扱う
- `editor-state.model_presets` と `editor-state.memory_sets` は編集対象なので、実値の `api_key` を含む shape で扱う

request body の最小 shape:

```json
{
  "current": {
    "selected_persona_id": "persona:default",
    "selected_memory_set_id": "memory_set:default",
    "selected_model_preset_id": "model_preset:default",
    "desktop_watch": {
      "enabled": false,
      "interval_seconds": 300
    },
    "wake_policy": {
      "mode": "disabled"
    }
  },
  "personas": [],
  "memory_sets": [],
  "model_presets": []
}
```

response は `GET /api/config/editor-state` と同じ shape を返す。

### `GET /api/config/personas/{persona_id}`

- 認証: 必要

### `DELETE /api/config/personas/{persona_id}`

- 認証: 必要
- 役割: 指定した `persona` を削除する

### `GET /api/config/memory-sets/{memory_set_id}`

- 認証: 必要
- 役割: 指定した `memory_set` の詳細を返す
- `memory_set` は人格設定と独立の記憶集合単位として扱う
- 読み取り response では `embedding.api_key` の実値を返さず、代わりに `api_key_present` を返す

### `PUT /api/config/memory-sets/{memory_set_id}`

- 認証: 必要
- 役割: 指定した `memory_set` を全体置換で作成または更新する
- 空の `memory_set` を新規作成する

request body の最小 shape:

```json
{
  "memory_set_id": "memory_set:default",
  "display_name": "Default Memory",
  "embedding": {
    "model": "openrouter/google/gemini-embedding-001",
    "embedding_dimension": 3072,
    "api_key": ""
  }
}
```

`embedding` は必要な場合だけ `api_base` を持つ。
`api_base` が未指定または空欄のときは、`model` の先頭から導ける既定の接続先を使う。
`embedding_dimension` は必須で、その `memory_set` の vector index 次元数を表す。
`embedding` を変更したときは、その `memory_set` の vector index を作り直す。

### `POST /api/config/memory-sets/clone`

- 認証: 必要
- 役割: 既存の `memory_set` を複製して新しい `memory_set` を作成する
- 複製後の `memory_set` は元の集合と独立して扱う
- 複製時は source 側の埋め込み設定も引き継ぐ

request body の最小 shape:

```json
{
  "source_memory_set_id": "memory_set:default",
  "memory_set_id": "memory_set:branch-a",
  "display_name": "Branch A Memory"
}
```

### `DELETE /api/config/memory-sets/{memory_set_id}`

- 認証: 必要
- 役割: 指定した `memory_set` 全体を削除する
- 個別の `event`、`episode`、`memory_unit` を削除する標準 API は持たない

### `PUT /api/config/model-presets/{model_preset_id}`

- 認証: 必要
- 役割: 指定したモデルプリセット定義を全体置換する

request body の最小 shape:

```json
{
  "model_preset_id": "model_preset:default",
  "display_name": "Default OpenRouter Gemini Preset",
  "prompt_window": {
    "recent_turn_limit": 30,
    "recent_turn_minutes": 3
  },
  "roles": {
    "observation_interpretation": {
      "model": "openrouter/google/gemini-3.1-flash-lite-preview",
      "api_key": "",
      "reasoning_effort": "low",
      "max_output_tokens": 3000,
      "web_search_enabled": false
    },
    "decision_generation": {
      "model": "openrouter/google/gemini-3.1-flash-lite-preview",
      "api_key": "",
      "max_output_tokens": 3000,
      "web_search_enabled": false
    },
    "expression_generation": {
      "model": "openrouter/google/gemini-3.1-flash-lite-preview",
      "api_key": "",
      "max_output_tokens": 4000,
      "web_search_enabled": false
    },
    "memory_interpretation": {
      "model": "openrouter/google/gemini-3.1-flash-lite-preview",
      "api_key": "",
      "max_output_tokens": 3000,
      "web_search_enabled": false
    },
    "memory_reflection_summary": {
      "model": "openrouter/google/gemini-3.1-flash-lite-preview",
      "api_key": "",
      "max_output_tokens": 2000,
      "web_search_enabled": false
    }
  }
}
```

`prompt_window` は会話履歴の取り込み窓である。
`recent_turn_limit` は件数上限、`recent_turn_minutes` は時間窓の上限を表す。

各 role は必要な場合だけ `api_base` を持つ。
`provider` は独立項目として持たず、`model` の先頭から導く。
`api_base` が未指定または空欄のときは、`model` の先頭から導ける既定の接続先を使う。

`reasoning_effort` は指定がある場合だけ保存する。空文字列や空白のみの値は保存時に削除し、未指定として扱う。
`max_output_tokens` は各 role で必須とする。
`web_search_enabled` は各 role で必須とする。`true` のときは provider 側の検索機能を request に渡すが、OtomeKairo 自身が検索処理を持つわけではない。

response:

```json
{
  "ok": true,
  "data": {
    "model_preset": {}
  }
}
```

- 認証: 必要
- 役割: 指定した `model_preset` の詳細を返す
- 読み取り response では `roles.*.api_key` の実値を返さず、代わりに `api_key_present` を返す

response の最小 shape:

```json
{
  "ok": true,
  "data": {
    "model_preset": {
      "model_preset_id": "model_preset:default",
      "display_name": "Default OpenRouter Gemini Preset",
      "prompt_window": {
        "recent_turn_limit": 30,
        "recent_turn_minutes": 3
      },
      "roles": {
        "decision_generation": {
          "model": "openrouter/google/gemini-3.1-flash-lite-preview",
          "max_output_tokens": 3000,
          "web_search_enabled": false,
          "api_key_present": true
        }
      }
    }
  }
}
```

## 主な失敗

| HTTP | `error.code` | 意味 |
|------|--------------|------|
| `400` | `invalid_wake_policy` | `wake_policy` が object ではない |
| `400` | `invalid_wake_policy_mode` | `wake_policy.mode` が不正 |
| `400` | `invalid_interval_minutes` | `wake_policy.interval_minutes` が不正 |
| `400` | `invalid_desktop_watch` | `desktop_watch` が object ではない |
| `400` | `invalid_desktop_watch_enabled` | `desktop_watch.enabled` が boolean ではない |
| `400` | `invalid_desktop_watch_interval_seconds` | `desktop_watch.interval_seconds` が不正 |
| `400` | `invalid_persona_display_name` | `persona.display_name` が空または文字列ではない |
| `400` | `invalid_persona_prompt` | `persona.persona_prompt` が空または文字列ではない |
| `400` | `invalid_expression_addon` | `persona.expression_addon` が文字列ではない |
| `400` | `unsupported_persona_field` | `persona` に未対応の追加フィールドが含まれる |
| `400` | `invalid_memory_set_display_name` | `memory_set.display_name` が空または文字列ではない |
| `400` | `invalid_embedding_definition` | `memory_set.embedding` が object ではない |
| `400` | `invalid_embedding_model` | `memory_set.embedding.model` が空または文字列ではない |
| `400` | `invalid_embedding_api_base` | `memory_set.embedding.api_base` が文字列ではない |
| `400` | `invalid_embedding_api_key` | `memory_set.embedding.api_key` が文字列ではない |
| `400` | `invalid_embedding_dimension` | `memory_set.embedding.embedding_dimension` が正の整数ではない |
| `400` | `invalid_prompt_window` | `model_preset.prompt_window` が object ではない |
| `400` | `invalid_recent_turn_limit` | `prompt_window.recent_turn_limit` が正の整数ではない |
| `400` | `invalid_recent_turn_minutes` | `prompt_window.recent_turn_minutes` が正の整数ではない |
| `400` | `missing_model_role` | 必須 role が不足している |
| `400` | `invalid_model_role` | role 定義が object ではない |
| `400` | `invalid_model_role_model` | `roles.*.model` が空または文字列ではない |
| `400` | `invalid_model_role_api_base` | `roles.*.api_base` が文字列ではない |
| `400` | `invalid_model_role_api_key` | `roles.*.api_key` が文字列ではない |
| `400` | `invalid_reasoning_effort` | `roles.*.reasoning_effort` が文字列ではない |
| `400` | `invalid_max_output_tokens` | `roles.*.max_output_tokens` が正の整数ではない |
| `400` | `invalid_web_search_enabled` | `roles.*.web_search_enabled` が boolean ではない |
| `400` | `invalid_editor_state_current` | `editor-state.current` が object ではない |
| `400` | `missing_personas` | `editor-state.personas` が空 |
| `400` | `missing_memory_sets` | `editor-state.memory_sets` が空 |
| `400` | `missing_model_presets` | `editor-state.model_presets` が空 |
| `404` | `persona_not_found` | 指定した `persona_id` が存在しない |
| `404` | `memory_set_not_found` | 指定した `memory_set_id` が存在しない |
| `404` | `model_preset_not_found` | 指定した `model_preset_id` が存在しない |
| `404` | `source_memory_set_not_found` | 複製元 `memory_set_id` が存在しない |
| `409` | `selected_persona_delete_forbidden` | 選択中の人格設定は削除できない |
| `409` | `selected_memory_set_delete_forbidden` | 選択中の記憶集合は削除できない |
| `409` | `selected_model_preset_delete_forbidden` | 選択中のモデルプリセットは削除できない |
| `409` | `memory_set_id_already_exists` | 複製先 `memory_set_id` が既に存在する |
| `409` | `last_resource_delete_forbidden` | 最後の定義は削除できない |
