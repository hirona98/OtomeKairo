# 用語表

この文書は、OtomeKairo と CocoroConsole で使う用語を統一するための参照表である。
意味規則の正本は各 design 文書に置き、この文書は正規語、wire/code 表記、避ける表記、使い分けを示す。

## 運用原則

- 日本語の UI、コメント、docs では正規語を使う。
- HTTP API、保存データ、識別子、実装上の field 名では wire/code 表記を使う。
- CocoroConsole は OtomeKairo の外部接点であり、OtomeKairo の設計語に合わせて表示文言とコメントを揃える。
- 既存の wire/code 表記は、API 仕様、保存形式、実装上の識別子として定義済みの場合だけ維持する。
- 古い名称や別概念の名称は、新規 docs、UI 表示、コメントに使わない。
