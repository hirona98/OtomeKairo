# AGENTS.md

## このリポジトリで開発するもの
- LLMを使用して人間を模した自立可能な個
- 継続的に存在し、観測し、理解し、判断する
- 会話AIではない


## 本システムの考え方
- 人格設定は、どう考え、どう振る舞い、何を重視するかの基底である
- 記憶は、何を経験し、何を理解し、誰や世界とどう関わってきたかの蓄積である
- 現在の個は、人格設定と記憶からその時点で成立する判断主体である

実際の挙動は、人格設定と記憶から組み上がる現在の個の文脈で決まる
人格設定だけでも、記憶だけでも、OtomeKairo の現在の個は定まらない


## 方針
- ユーザーとのコミュニケーションには日本語を使う
- シンプルさを優先すること
- 要件や設計の判断は docs ディレクトリを正として扱うこと
- 同じ仕様を複数の docs に正本として書かないこと。意味境界、API wire、実装計画の正本を分け、他の docs ではリンク参照に留めること
- docs とコードは常に同期させること
- 修正時は、本来あるべき姿への修正を行うこと（最小限の修正にしない、ガードするのではなく根本から変更する）
- 旧仕様との互換レイヤーや移行用コードを入れないこと
- フォールバック処理を入れないこと
- 仕様が固まるまでは、新規テストは原則として実装しない
- コメントを書くこと
- ログ、サンプル、テストデータに秘密情報を含めない


## ツール運用
- Gitのユーザ名とメールアドレスを変更しない
- ユーザーが明示的に指示するまでコミットしない
